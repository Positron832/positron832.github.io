Cool HTML trix

So I'm not a fan of the "/index.html" in the URL, so whenever I link links to my main page I always write the full URL. However, yesterday I discovered that you can just write "/" in the href attribute and the link takes you back to my main page without showing "/index.html".

Another tip for people doing XHTML syntax is to use <?xml-stylesheet href="..." etc?> at the top at the document, after the <?xml version stuff. The reason I recommend linking to the CSS file that way is because with the normal way you get a splash of unstyled content (it's very noticeable with files using the .xml file extension).

Well that's it! Might add more trix in a future blog. 17th Feb '21
